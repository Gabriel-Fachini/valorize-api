# Brainstorm de features

Esse documento é um resumo de features pensadas em um brainstorm para esse projeto. Algumas delas já estão implementadas e outras não.

## Categorias

- Elogios
- Prêmios
- Ressgates
- Transações
- Configurações
- Noticias
- Lives/Eventos
- Iniciativas
- Educação
- Treinamentos

### Elogios

- [x] Um usuário pode fazer um elogio ao outro
- [x] Cada elogio deve ser associado à um valor da empresa
- [ ] Reações aos elogios

#### Elogio Admin

- [ ] Configurar valores da empresa
- [ ] Dados relevantes como quantidade de elogios trocados, quantidade de moedas movimentdas, qual valor está aparecendo mais nos elogios, índice de engajamento de elogio (quantas pessoas usaram em determinado tempo)
- [ ] Identificação de trocar de elogios e moedas (Por IA ou outro método)
- [ ] Configurar quantidade de renovação do saldo semanal
- [ ] Identificação de clima pelas mensagens com analise feita por IA

### Prêmios

- [x] Visualizar catálogo de prêmios com filtros
- [x] Visualizar detalhes de um prêmio com possibilidade de escolher variante e ver número de unidades disponíveis em estoque
- [x] Possibilidade de escolher endereço de envio ou criar um novo na hora de confirmar o resgate

#### Prêmios Admin

- [ ] Deve ser possível visualizar analytics como:
- Prêmios mais resgatados
- Categoria mais resgatada
- Quantidade gasta
- N de resgates
  
### Resgates

- [x] Visualizar detalhes de resgate de prêmios
- [x] Acompanhar Rastreamento da entrega
- [x] Cancelar resgate em até 24h

#### Resgates Admin

- [ ] Visualizar todos os resgates com código de rastreio
- [ ] Analytics de resgate
- Tempo de entrega médio, número de falhas
- Estatísticas úteis para gestores

### Transações

- [x] Visualizar detalhes das transações com auditoria de saldo anteiror e após a movimentação
- [x] Filtrar registros
- [x] Mostrar id da transação para suporte e debug

#### Transações Admin

Não foi pensado nenhuma funcionalidade interessante para o admin nessa parte.

- [ ] Comentar notícias

#### Notícias Admin

- [ ] Analytics relevantes
